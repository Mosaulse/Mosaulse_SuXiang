# Su Xiang — A Classical Color Theme for VSCode

*A scroll of Su Xiang, cast upon your code — the hue of an aged manuscript, woven into every line.*

---

## Origin

> *Bundles of silk scrolls, too precious to open lightly.*
> — *Yan's Family Instructions (颜氏家训 · 勉学)*

**Su Xiang** is a `VSCode` color theme drawn from the aesthetics of ancient Eastern manuscripts, comprising **three series of six volumes**.

"**Su**" (素) denotes raw, undyed silk — the plainness before all adornment. As the *Book of Rites* records: "A great officer wears a plain sash." Su is the ground, the blank space, the quietude before ink meets paper.

"**Xiang**" (缃) denotes pale amber silk — the hue of aged paper. As the *Shuowen Jiezi* defines it: "Xiang, the pale yellow of silk." It is the color of a scroll's flyleaf, a seal of time itself — pages yellowed by years, that warm amber glow the first annotation written by passing ages.

This theme harbors no harsh fluorescent whites, nor abyssal blacks. Only the warmth of aged paper, the faded ink of settled time, the deliberate impression of a vermillion seal, and the serene jade-green of celadon headings. As though unrolling an ancient scroll in the afternoon sun or by flickering candlelight — all code becomes hushed and warm.

---

## Three Series, Six Volumes

Rooted in the [Chinese Traditional Color Spectrum](https://zhongguose.com/), this theme comprises three series — **Suxiang**, **Zixiang**, and **Fuqu** — each with a light and dark volume, forming six distinct atmospheres.

### Suxiang Series — Fragrance of Antique Scrolls

Colors drawn from ancient book ink and paper. Raw silk as the ground, vermillion to punctuate, jade green to outline — like unrolling a Song-dynasty woodblock print, the scent of ink filling the room.

### Zixiang Series — Purple Smoke by Candlelight

Colors drawn from twilight candlelight and purple smoke. Deep violet shadows, pink paper like dawn — like reading by candlelight, code flowing silently through purple mist.

### Fuqu Series — Lotus upon Clear Water

Colors drawn from lotus blossoms and clear water. Moonlight casts shadows, fragrance drifts — like a summer evening by the pond, code cool as water.

## 📜 Installation

> *Unroll the Su Xiang scroll, and let your eyes rest.*
> *Settle in, and the meaning comes of itself.*

**Method 1: Install within VSCode**
1. Open VSCode
2. Click the **Extensions** icon in the sidebar (`Ctrl+Shift+X`)
3. Search for **Su Xiang** or **素缃**
4. Click **Install** and wait for the ink to dry
5. Press `Ctrl+K Ctrl+T` (Mac: `Cmd+K Cmd+T`), and choose any of the six volumes

**Method 2: Marketplace**

Or find this theme directly on the [VSCode Marketplace](https://marketplace.visualstudio.com/) — one search, one click.

## ⚙️ Recommended Configuration

> *A craftsman must first sharpen his tools.*
> — *The Analerta (论语 · 卫灵公)*

For the best classical experience, add these settings to your `settings.json`:

```json
{
  "editor.fontFamily": "'LXGW WenKai Mono', 'Maple Mono NF CN'",
  "editor.fontSize": 15,
  "editor.lineHeight": 1.6,
  "editor.renderWhitespace": "selection",
  "editor.bracketPairColorization.enabled": true,
  "editor.guides.bracketPairs": true
}
```

> **Special Recommendation**: [**LXGW WenKai (霞鹜文楷)**](https://github.com/lxgw/LxgwWenKai) — an open-source Chinese font based on Klee One, blending the beauty of Kaishu and Fangsong typefaces. Warm, elegant, and perfectly complementing the Su Xiang theme.

## 📚 Language Support

> *A single scroll of Su Xiang speaks a thousand tongues.*
> *Any language becomes poetry, any code becomes painting.*

This theme relies on standard TextMate semantic tokenization, enhanced with fine-grained `semanticTokenColors` configuration. Especially well-adapted languages include:

**Frontend Trio**: JavaScript · TypeScript · HTML/CSS
**Backend Classics**: Python · Java · C# · Go · Rust
**Data Languages**: SQL · JSON · Markdown
**System Voices**: C/C++ · Shell

## 🛠️ Local Development

> *To study alone without friends is to be narrow and ill-informed.*
> — *Book of Rites · Record of Learning (礼记 · 学记)*

To explore this theme in depth and savor every hue:

```bash
# Clone the scroll
git clone https://github.com/mosaulse/suxiang-theme.git
cd suxiang-theme

# Install the scholar's tools
npm install

# Open VSCode and press F5 to launch an Extension Development Host
code .

# Package for distribution
npx vsce package
```
